# 名前　jyugemu

コマンド
+ jyugemu start 起動
+ exit
+ jyugemu init　jyugemu.jsonをルートに作成

できること
+ コマンドラインで打ったコマンドを記録できる
+ コマンドを打った日時を記録できる
+ コマンドを打った人を記録できる
+ 記録したコマンドをサジェストできる
+ ignoreで特定のコマンドの使用を禁止できる
+ envを読み込ませて環境変数を書かせない

✨ jyugemu.json を作成しました。
⚠️  [重要] 事故防止のため、.gitignore に以下を追記することをお勧めします：

    jyugemu.json

作業を開始するには 'jyugemu start' を叩いてください。
